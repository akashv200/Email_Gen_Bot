# Quick Start Guide - Email Generator Bot

## ‚ö° Get Started in 3 Minutes

### For Windows Users (Recommended)

#### Option 1: Run Directly (Easiest)

1. **Install Python** (one-time setup)
   - Go to https://www.python.org/downloads/
   - Download Python 3.7 or higher
   - Run installer
   - ‚úÖ **IMPORTANT**: Check "Add Python to PATH"

2. **Download the Application**
   - Download `email_generator_bot.py`
   - Save it anywhere (e.g., Desktop or Documents)

3. **Run the Application**
   - Double-click `email_generator_bot.py`
   - Or open Command Prompt and type:
     ```
     python email_generator_bot.py
     ```

That's it! The application will open and you can start generating emails immediately.

---

#### Option 2: Create Windows Executable (Advanced)

1. **Install Python** (if not already done)
   - Same as Option 1 above

2. **Download Files**
   - Download `email_generator_bot.py`
   - Download `setup.py`
   - Put them in the same folder

3. **Run Setup**
   - Double-click `setup.py` or run:
     ```
     python setup.py
     ```
   - Choose option 1 (Build executable)
   - Wait for build to complete

4. **Run Your Executable**
   - Find `EmailGeneratorBot.exe` in the `dist` folder
   - Double-click to run
   - No Python needed anymore!

## üêõ Troubleshooting

### Problem: Python not found

**Error**: "Python is not recognized..."

**Solution**:
1. Install Python from python.org
2. During installation, check "Add Python to PATH"
3. Restart your computer
4. Try again

### Problem: Application won't start

**Error**: Nothing happens when double-clicking

**Solution**:
1. Right-click the file
2. Select "Open with" ‚Üí "Python"
3. Or open Command Prompt:
   ```
   cd path\to\file
   python email_generator_bot.py
   ```

### Problem: Tkinter not found

**Error**: "No module named 'tkinter'"

**Solution**:
- Reinstall Python
- During installation, ensure "tcl/tk and IDLE" is checked
- Complete installation
- Try again

### Problem: Window disappears immediately

**Solution**:
- There's an error in the code
- Run from Command Prompt to see error:
  ```
  python email_generator_bot.py
  ```
- Error message will show what's wrong
